# **App Name**: Dashboard

## Core Features:

- Quote Requests: Clients can request project quotes with detailed specifications.
- Live Messaging: Real-time chat for developers and clients to discuss project details.
- Progress Tracking: Visual representation of project milestones and task completion.
- Notifications: Alerts for new messages, updates, and completed tasks.
- Role-Based Access: Secure access controls to protect sensitive information.
- GitHub Automation: Automated progress updates from development activity on GitHub. The tool is also aware of GitHub links. Links back to code.
- Settings: Dark and Light mode toggle, and option to change contact email (login email remains the same)

## Style Guidelines:

- Primary color: Dark blue (#243A73) for professionalism and trust.
- Background color: Very light desaturated blue (#F0F4FF).
- Accent color: Purple (#6A459E) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif, for a clean and modern look.
- Simple and intuitive icons for navigation and actions.
- Clean and structured layout with clear sections for each feature.
- Subtle transitions and animations to enhance user experience.
